# 생산발주량 대시보드 — 사내 공유 서버 사용법

## 구조 이해하기 (중요)

- 이 패키지를 실행하는 PC 1대가 **서버 역할**을 합니다.
- 같은 사내망(같은 Wi-Fi/공유기/회사 LAN)에 있는 다른 PC들은
  서버 PC의 IP 주소로 접속해서 **화면을 봅니다(조회 전용)**.
- 데이터(엑셀 업로드, IndexedDB 저장)는 **서버 PC의 브라우저에만** 저장됩니다.
  → 데이터 업로드/입력 작업은 **반드시 서버 PC에서** 하세요.
  → 다른 PC에서 접속한 화면도 같은 데이터를 "조회"는 할 수 있지만,
     그 PC에서 새로고침하면 서버가 보여주는 최신 화면을 다시 받아옵니다.

## 1. 서버 PC에서 실행하기

### Windows
1. `start_server.bat` 파일을 더블클릭합니다.
2. 검은 창(터미널)이 열리고, 접속 주소가 안내됩니다.
   ```
   [1] 이 PC(서버)에서 직접 접속:
       http://localhost:8080
   [2] 다른 PC에서 접속할 주소:
       http://192.168.0.15:8080   ← 예시. 실제로는 다른 숫자가 표시됩니다.
   ```
3. 이 창은 **닫지 마세요**. 창을 닫으면 서버가 꺼지고 다른 PC에서 접속이 끊깁니다.
4. 최초 실행 시 Windows 방화벽 경고가 뜨면 **"허용"**을 눌러주세요.

### macOS / Linux
1. 터미널을 열고 이 폴더로 이동합니다.
2. 아래 명령을 한 번만 실행해 실행 권한을 부여합니다.
   ```
   chmod +x start_server.sh
   ```
3. 실행합니다.
   ```
   ./start_server.sh
   ```

## 2. 다른 PC에서 접속하기

1. 서버 PC 화면에 안내된 주소(예: `http://192.168.0.15:8080`)를 확인합니다.
2. 같은 사내망에 연결된 PC의 웹브라우저(Chrome, Edge 등) 주소창에 그대로 입력합니다.
3. 대시보드 화면이 그대로 뜹니다.

## 3. 자주 발생하는 문제

**Q. 다른 PC에서 접속이 안 됩니다.**
- 서버 PC와 다른 PC가 **같은 네트워크**(같은 사내 LAN, 같은 와이파이)에 있는지 확인하세요.
- 서버 PC의 Windows 방화벽에서 포트 8080이 막혀있을 수 있습니다.
  `start_server.bat`을 **마우스 우클릭 → 관리자 권한으로 실행**하면 방화벽 규칙이 자동으로 추가됩니다.
- 사내망에 별도 보안 솔루션(망분리, NAC 등)이 있다면 IT 담당자에게 포트 8080 허용을 요청하세요.

**Q. IP 주소가 바뀌어서 접속이 안 됩니다.**
- 사내 PC는 보통 IP가 자동으로 바뀌지 않지만(고정IP 또는 DHCP 예약),
  바뀌었다면 서버 PC에서 `start_server.bat`을 다시 실행해 새로 안내되는 주소를 확인하세요.
- 매번 바뀌는 게 불편하면 IT 담당자에게 서버 PC의 IP를 고정으로 설정해달라고 요청하는 것을 권장합니다.

**Q. 서버 PC에서 엑셀을 업로드했는데 다른 PC 화면에 안 보여요.**
- 다른 PC에서 브라우저 새로고침(F5)을 해보세요.
- 단, 이 대시보드의 데이터 저장 방식(브라우저 IndexedDB) 특성상,
  다른 PC가 접속한 시점에 따라 화면에 보이는 데이터가 약간 다를 수 있습니다.
  데이터의 "원본"은 항상 서버 PC의 브라우저에 있다고 생각하시면 됩니다.

## 4. 포트 번호를 바꾸고 싶다면

`start_server.ps1` (Windows) 또는 `start_server.sh` (macOS/Linux) 파일 상단의
`$port = 8080` 또는 `PORT=8080` 부분의 숫자를 원하는 포트로 바꾸면 됩니다.
(이미 다른 프로그램이 8080을 쓰고 있을 때만 바꾸시면 됩니다.)

## 파일 구성

| 파일 | 용도 |
|---|---|
| `index.html` | 생산발주량 대시보드 본체 |
| `start_server.bat` | Windows에서 더블클릭으로 서버 실행 |
| `start_server.ps1` | Windows PowerShell 서버 스크립트 (bat이 내부에서 호출) |
| `start_server.sh` | macOS/Linux용 서버 실행 스크립트 |
